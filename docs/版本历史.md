
# V3.4.0 升级内容：2020-03-08
- 查询条件值进行类型转换处理，避免类型不匹配异常；
- 增加IN查询方式的支持；
- 新增对数字类型：byte，short，BigInteger，BigDecimal的支持；
- 修正：区间范围查询如果结束值小于开始值查不到数据的问题；
